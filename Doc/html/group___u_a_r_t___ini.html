const uint32_t UART1_Init ( STM32F051_UART_t *const  pUART)

‪Initialization of the UART1 module and its periphery.

The function adjusts the pins of the PA9 and PA10 microcontroller to work with the UART1 module, configures the module itself with the following parameters: 

Speed:     9600
Parity:    None
Data bits: 8
Stop bits: 1
Parameters
[in,out]pUART‪Pointer to object of UART1 module
Returns
‪UART1 initialization status
Return values
0‪If initialization was successful.
1‪If an initialization error occurs.
